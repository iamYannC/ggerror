# ggerror: Extended Error Geoms for 'ggplot2'

Extends the 'ggplot2' error geoms. geom_error() accepts an 'error'
aesthetic with auto-inference of the orientation. It also supports
'error_neg' and 'error_pos' for asymmetric cases, with full control over
aesthetics per side, such as color, width etc... The package also
includes a vignette covering it's main usecases - symmetric, asymmetric,
one-sided, and per-side styling.
